METADATA EXTRACTOR

This is application to extract metadata from pdf and load it to GeoNetwork.

You can edit and configure data in iso19115v2/FGDC format.

Program extract:
- authors
- title
- year
- names
- locations with geografic coordinats
- bibleografic references
- kew words
- key phrases
